# Temporial

The 2D-to-4D engine — turning a single image into a clean, rigged, animated 3D model.
A co-founder invitation site.

## View
Open `index.html` (or `temporial-cinematic.html`) in a browser. The page is self-contained
(3D models embedded); it needs internet for fonts and the model-viewer library.

## Structure
- `index.html` / `temporial-cinematic.html` — the site
- `assets/` — videos, images, 3D models, and the scroll-background video
